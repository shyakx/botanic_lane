### Adding Products
Edit `src/data/mockData.js` to add new products:
```javascript
{
  id: 9,
  name: "Your Product Name",
  price: 29.99,
  image: "your-image-url",
```

- Component styles: `src/App.css`
- Color variables defined in CSS custom properties
